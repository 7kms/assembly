# 汇编语言笔记

本笔记的核心脉络是王爽老师的`《汇编语言(第三版)》`

学习目的:
1. 获得底层编程的体验
2. 深刻理解计算机运行程序的机理

王爽老师的书籍是在`8086CPU`上编写的. 纯粹的`8086CPU`现在已经不存在了, 但是任何一台和Intel系列兼容的pc机都可以8086的方式进行工作(可以当做一台高速的8086PC)
